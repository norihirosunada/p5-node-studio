# Project Overview

## Project Name
p5-node-studio (Folder: nodegraph-visualizer)

## Purpose
A node graph style interface for p5.js coding, allowing users to build generative art and graphics with a modular, visual workflow. It features real-time preview, live code editing, and dynamic parameters.

## Tech Stack
- **Framework**: React 19, TypeScript
- **Build Tool**: Vite
- **Graphics Engine**: p5.js
- **Styling**: Tailwind CSS (as per README)
- **Icons**: Lucide React
- **Environment**: Node.js

## Key Features
- Node-Based Workflow
- Real-Time Preview
- Live Code Editing
- Dynamic Parameters
- Modular Design

## Structure
The project seems to have a flat structure with source files in the root directory:
- `App.tsx`, `index.tsx`: Entry points
- `components/`: React components
- `utils/`: Utility functions
- `services/`: Service layers
- `assets/`: Static assets
- `types.ts`: TypeScript definitions
- `constants.tsx`: Constants

## Configuration
- `vite.config.ts`: Vite configuration (port 3000, alias `@` to root)
- `tsconfig.json`: TypeScript configuration (ES2022, React JSX, strict-ish)
